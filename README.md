## IF_testes 🪚

Aqui tive a oportundidade de testar o IF no meu programa, onde no arquivo "IF_" realizei uma mudança do projeto anterior, onde apenas dizia o IMC da pessoa, porém nesse atual ele determina se seu IMC está dentro da faixa padrão de peso, ou está com irregularidades, no próximo já é mais simples, onde determina se o número dividido por 2, sobra resto ele será Ímpar, caso contrário, será Par.
